Runtime MCA world data copied from the Minecraft server.

`MCA-FamilyTree.dat` lets the Render proxy answer NPC family questions about
parents, children, spouse/partner, gender and alive/dead status. Replace this
file with a fresh copy from `world/data/MCA-FamilyTree.dat` after important MCA
family changes in the Minecraft world.

Do not put API keys, tokens, logs or memory databases in this folder.
